Introduction

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It structures the content on the web and defines elements like headings, paragraphs, links, images, and more. HTML works alongside CSS and JavaScript to build dynamic and visually appealing websites.

Basic Syntax

HTML documents consist of elements enclosed in tags. Tags are written inside angle brackets < > and usually come in pairs: an opening tag <tag> and a closing tag </tag>. Some tags are self-closing.

Structure

standard structure of an HTML document :

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Welcome to HTML</h1>
    <p>This is a simple HTML page.</p>
  </body>
</html>

Common HTML Tags

Document Structure

  • <html>: Root element of an HTML document.
  • <head>: Contains metadata (title, links to CSS/JS files, etc.)
  • <title>: Sets the title of the web page (shown in the browser tab).
  • <body>: Contains the main content of the web page.

Content Tags

  • <h1> to <h6>: Headings, where <h1> is the largest, and <h6> is the smallest.
  • <p>: Paragraph.
  • <br>: Line break (self-closing).
  • <hr>: Horizontal line (self-closing).
Best Practices
  • Indentation and Formatting: Use consistent indentation to make the code readable.
  • Comments: Add comments using <!-- --> for clarity and to explain complex sections.
  • Doctype Declaration: Always start with <!DOCTYPE html> to specify the HTML version.
  • Semantic Tags: Use semantic tags like <header>, <footer>, <article>, and <section> for better accessibility and SEO.
  • Alt Text for Images: Always include descriptive alt attributes for images to improve accessibility.
  • Avoid Deprecated Tags: Avoid using outdated tags like <font> or <center>. Use CSS instead.
